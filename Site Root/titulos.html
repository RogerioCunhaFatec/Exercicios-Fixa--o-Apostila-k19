K02 - Desenvolvimento Web com HTML, Css e Javascrip

Atualmente, praticamente todos os sistemas corporativos possuem interface web. Paara que deseja atuar no mercado de desenvolvimento de software, é obrigatório o conhecimento das linguagens: HTML, CSS e Javascrip

Essas linguagens são utilizadas para o desenvolvimento da camada de apresentação das aplicações web.
